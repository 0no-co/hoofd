# react-head

This project aims at providing a set of hooks to populate `<meta>`, ... for each page.

## Goals

Initially this will be `title` and `meta`.

- [x] React support
- [x] Add majority of types for meta and link.
- [x] Concurrent friendly
- [x] Preact support
- [x] Support `<link>`
- [ ] Stricter typings
- [ ] SSR support
- [ ] Document the dispatcher
- [ ] Document the hooks
- [ ] Golf bytes
